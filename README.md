## 介绍
mid 框架脚手架

## 安装
- 需要先全局安装pm2 (sudo npm i pm2 -g)
- sudo npm i mid-cli -g

## 命令
### 帮助
`
mid-cli -h
`
### 查看版本
`
mid-cli -v
`
### 初始化
`
mid-cli init
`
